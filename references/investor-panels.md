## DRAGONFLY
*"Deeply technical crypto-natives. Researchers and builders who push the ecosystem forward."*

- **Their focus**: Technical depth and credibility. Protocol economics that actually work. Defensibility in open-source environments. Global adoption thesis (US + Asia lens). Checks $3M–$30M+.
- **Strongest card**: [what resonates]

## 1KX
*"Architecting network success. Investing at key inflection points."*

- **Their focus**: Is this a genuine inflection-point moment for blockchain? Token network design quality — is the token structurally necessary or bolted on? How do you solve the cold-start problem? Where's the network tipping point?

## Common Narrative Gap Patterns by Investor Type

| Investor type | What they will probe |
|--------------|---------------------|
| Generalists (Sequoia, a16z) | Unit economics hand-wavy; unclear why you win vs. well-funded incumbent; team credential gap for market size |
| Crypto-native (Dragonfly, 1kx) | Token design not defensible; no answer to fork risk; cold-start problem unaddressed |
| Consumer-focused (Archetype, Outlier) | No mainstream UX path; crypto-native only; community mechanics unclear |
| Long-cycle (Blockchain Capital, Borderless) | Cycle dependency; no institutional-grade properties; decentralization theater |
